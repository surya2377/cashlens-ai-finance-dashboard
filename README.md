## ✨ Features

- 📊 **Instant Analysis** - Upload CSV/PDF bank statements, get insights in seconds
- 🤖 **AI Categorization** - Automatic transaction categorization using Google Gemini
- 💬 **Chat with Your Finances** - Ask questions in plain English, get instant answers
- 🔍 **Subscription Detection** - Automatically finds recurring charges you might have forgotten
- 💸 **Money Leak Alerts** - Identifies small frequent expenses that add up
- 📈 **Spending Forecast** - AI predicts next month's expenses based on trends
- 📋 **CFO Executive Report** - Professional financial summary generated by AI
- 🎯 **Financial Health Score** - Get rated 0-100 based on savings, volatility, and habits

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/cashlens-ai-finance-dashboard.git
cd cashlens-ai-finance-dashboard

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Add your Gemini API key to .env
# VITE_GEMINI_API_KEY=your_api_key_here

# Start development server
npm run dev
```

## 🔑 Getting an API Key

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create a new API key
3. Add it to your `.env` file as `VITE_GEMINI_API_KEY`

## 🎯 Usage

1. **Upload** - Drop your bank statement (CSV format with date, description, amount columns)
2. **Analyze** - AI processes transactions in 15-30 seconds
3. **Explore** - Navigate through Overview, Transactions, Subscriptions, Insights, Projections
4. **Chat** - Ask questions like "Where did I waste the most money?" or "How much did I spend on food?"

## 📋 CSV Format

Your bank statement should have these columns:
```csv
date,description,amount
2026-01-15,Netflix Subscription,-15.99
2026-01-16,Salary Deposit,3500.00
2026-01-17,Grocery Store,-67.43
```

- **Date:** YYYY-MM-DD format
- **Description:** Transaction description
- **Amount:** Negative for expenses, positive for income
